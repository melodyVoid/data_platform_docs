# 组件

src/components 用于存放一些公用的业务组件，在 element-ui 的基础上结合业务进行的二次封装。

刚开始组件少的时候是一个一个单独注册，随着组件越来越多，main.js 里注册一堆全局组件会显得很臃肿。所以给公用组件添加一个自定义组件的入口文件，以插件的形式进行注册。

src/components/index.js 为入口文件，里面进行全局注册组件，并在最后导出 install 方法。

import SvgIcon from './SvgIcon'
import RadioGroup from './RadioGroup'
import ExportTable from './ExportTable'
import UserType from './UserType'
import SkuType from './SkuType'
import TimeCategory from './TimeCategory'
import DateRange from './DateRange'
import CheckboxGroup from './CheckboxGroup'
import MonthRange from './MonthRange'
import MultTable from './MultTable'
import CourseSearch from './CourseSearch'

const components = [
  SvgIcon,
  RadioGroup,
  ExportTable,
  UserType,
  SkuType,
  TimeCategory,
  DateRange,
  CheckboxGroup,
  MonthRange,
  MultTable,
  CourseSearch
]

const install = Vue => {
  components.map(component => Vue.component(component.name, component))
}

export default install

然后在 main.js 中引入并使用该插件进行全局注册组件。

import Components from 'src/components'

Vue.use(Components)
