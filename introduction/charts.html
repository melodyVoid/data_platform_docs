# 图表

图表这里用到了 v-charts。是基于 echarts 的封装。

在使用 echarts 生成图表时，经常需要做繁琐的数据类型转化、修改复杂的配置项，v-charts 的出现正是为了解决这个痛点。基于 Vue2.0 和 echarts 封装的 v-charts 图表组件，只需要统一提供一种对前后端都友好的数据格式设置简单的配置项，便可轻松生成常见的图表。

因为只用到了「折线图」、「柱状图」、「饼图」、「条形图」和「热力图」。所以在 src 文件夹下定义了 v-charts 文件夹来实现按需加载。

src/libs/v-charts/index.js 为入口文件，里面引入上述的几个图表组件，并全局注册，然后导出 install 方法。

import Vue from 'vue'
import VeLine from 'v-charts/lib/line.common' // 折线图
import VeBar from 'v-charts/lib/bar.common' // 条形图
import VeHeatmap from 'v-charts/lib/heatmap.common' // 热力图
import VePie from 'v-charts/lib/pie.common' // 饼图
import VeHistogram from 'v-charts/lib/histogram.common' // 柱状图

const components = [
  VeLine,
  VeBar,
  VeHeatmap,
  VePie,
  VeHistogram
]

components.map(component => Vue.component(component.name, component))

然后在 src/libs/index.js 中引入

import './v-charts'
