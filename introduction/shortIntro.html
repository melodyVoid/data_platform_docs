# 简介

混沌数据平台是将数据以一种可视化的形式表现出来的项目。主要有七大模块：

1. 用户数据

  • 用户画像
  • 新增激活
  • 付费统计

2. 首播

  • 整体统计
  • NPS 评分
  • 到/完课率

3. 回放

  • 整体统计
  • 课程列表
  • 分布情况

4. 传播转化

  • 自然流量转化率
  • 分享抢课
  • 病毒系数

5. 研值

  • 整体统计
  • 分布区间
  • 获取消耗

6. 学分/学位

  • 学分/学位分布

7. 续费率

  • 整体统计

技术栈： vue + vue-router + axios + vuex

UI： element-ui

图表： v-charts

脚手架： @vue/cli 3 选项：Babel、Router、Vuex、CSS Pre-processors(Sass/SCSS)、Linter / Formatter(ESLint + Prettier)

项目用 class 的形式代替了对象形式的书写，借助了 vue-class-component、vue-property-decorator、vuex三个库的支持。
