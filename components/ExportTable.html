    # 可导出的表格

    # 描述 [Description]

    表格组件，附带导出功能。表格左上角有导出按钮，用于导出 excel。表格数据超过5条（默认值，可设置）会自动折叠，并在表格底部出现"点击展开全部 >>"按钮。展开后点击"点击收起 <<"会收起表格。

    # 用法 [Usage]

    <export-table
  :table-data="tableData"
  :table-header="tableHeader"
  :filename="filename"
  :percent="['visit_rate', 'listen_rate']"
  :size="3"
></export-table>
